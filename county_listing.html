County Listing

This list provides links to each county and a description of what is currently available within the database and in some cases what is being worked on. Although most parishes listed are from the Anglican Church, FreeREG is also adding entries from non-conformist registers.

If you are interested in contibuting to FreeREG please follow the link to How to Volunteer for information on how to participate.

IMPORTANT - PLEASE NOTE

If you send an email to FreeREG via any of the links, then it is possible, because of the Spam filters, that your message may not get through. So if you do not get a reply within a week, please resend your message using these details, which will bypass the filters.

All counties use the old county areas.
